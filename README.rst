Spinete
========

A real-time sensor data visualization tool. `Have a look at it <https://www.youtube.com/watch?v=3zSo5SxCwp8>`_!


Launching Spinete
=================

Simply install the dependencies:

    pip install -r requirements.txt

And run Spinete:

    python main.py
